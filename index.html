Mau jadi web developer ?

Ini Roadmap nya untuk menjadi web developer

Sekarang adalah era digital dimana kebutuhan manusia bergantung kepada sebuah teknologi. Profesi progammer adalah salah satu yang sangat di butuhkan sekarang ini khususnya dalam pengembangan website. Web Developer, adalah seorang yang membuat atau mengembangkan sebuah website. Ini adalah salah satu pekerjaan yang menguntungkan. Karena pekerjaan ini di cari dimana - mana.

Tentu untuk menjadi web developer tidaklah mudah, karena untuk menjadi web developer kita tidak boleh berhenti belajar. Karena setiap tahun pasti ada update terbaru atau ada metode baru yang harus kita pelajari untuk meningkatkan kinerja kerja kita. Tetapi menjadi web developer ini sangat menguntungkan, apalagi sekarang era nya digital. Karena itu banyak yang meminati pekerjaan ini

Apa saja sih yang harus di persiapkan untuk menjadi web developer ?. Untuk memulai menjadi web developer kaliah hanya memperlukan sebuah device seperti laptop dan computer untuk ngoding. Yah hanya itu simple bukan. Peringatan jangan pernah untuk ngoding di handphone karena hal ini tidak efektif tetapi pilihan ada di tangan kalian. Ketika kalian sudah memiliki device yang kalian butuhkan, kalian bisa mulai ngoding.

Untuk web development ada dua jalur yang tersedia, yaitu front-end developer dan back-end developer. Sebenarnya ada juga yang namanya full-stack developer. Apa itu full-stack developer ?, full-stack developer adalah seorang web developer yang menguasai back-end dan front-end developer. Tetapi untuk menjadi full-stack developer ini tidak saya sarankan karena untuk mengikuti perkembangan teknologi front-end dan back-end itu tidak sempat, Walaupun ada progammer yang mengaku seorang full-stack developer. Pasti progammer ini fokus ke satu bidang saja, jadi progammer ini hanya menguasai salah satu bidang saja misalnya front-end. Walaupun ia bisa back-end biasa nya dia hanya menguasai basic nya saja. Jadi saya sarankan hanya mendalami satu bidang saja.

Nah sekarang bagaimana cara saya memilih bidang yang sesuai dengan saya ?. Baiklah saya akan menjelaskan apa yang harus bisa kalian kuasai untuk menjadi front-end dan back-end developer.

Untuk menjadi seorang front-end developer kalian harus memiliki skill mendesain dalam suatu website. front-end membutuhkan ke kreatifitas yang tinggi. Di sini kalian harus mengerti bagaimana struktuk website terbentuk. Kalian harus bisa menggunakan pewarnaan yang sesuai dan bagus.

beda bidang beda lagi skill yang dibutuhkan. Untuk menjadi seorang back-end developer kalian harus memiliki pemikiran yang kritis dan logic. Disini kalian harus menggunakan logika dalam menyelesaikan suatu masalah dan masih banyak lagi. Walaupun kalian bukan orang yang bisa berpikir kritis kalian bisa melatih nya, Sebaliknya front-end juga bisa di latih.

Permulaan

Saya akan jelaskan apa itu Back-end dan Front-end. Front-end adalah salah satu jalur web progamming yang mana ini memfokuskan kepada mendesain sebuah website. Front-end akan membuat tampilan website yang menarik. Back-end adalah jalur web progamming yang mengfokuskan kepada cara kerja suatu website. Back-end ini akan mengatur database dan lain - lainnya

Front-end

Pertama yang harus kalian pelajari adalah bagaimana internet bekerja, Apa itu HTTP ?, belajar mengenai browser dan bagaimana cara bekerjanya, dns, nama domain, dan apa itu hosting. Pokoknya untuk permulaan belajarlah mengenai internet dan cara bekerja nya.

Lalu mulai lah belajar HTML bahasa markup, Yang di gunakan untuk membuat struktur website, pelajari basic - basic nya. Lalu coba praktekan dengan menuliskan kode - kodenya. Pelajari mengenai form. Lalu latihan terus menerus. Nanti terbiasa sendiri dan hal ini akan membuat anda jago HTML dalam beberapa hari hanya dengan latihan menulis baris kode nya.

Masuk ke CSS, yaitu bahasa pemograman dimana digunakan untuk menghias sebuah website. Pelajari basic nya dan belajar membuat layouting dengan float, flex dan lain - lain nya.Jika sudah lancar dan bisa kalian bisa mulai mempelajari responsive design dan media queries. Agar website bisa di tampilkan di device lain selain Komputer dan laptop.

Ke tiga kalian bisa belajar JavaScript, beda dengan HTML dan CSS yang adalah bahasa markup. JavaScript ini adalah bahasa scripting, disaan kalian mempelajarinya kalian mungkin akan kesulitan karena konsep nya agak berbeda dari bahasa markup tapi jangan takut kalian tetap bisa mempelajarinya. Pelajarilah syntax dasar dan struktur sederhana nya terlebih dahulu. Lalu belajarlah DOM atau Document Object Model. Belajar Fetch API, ES6+, dan modular javascript. Pelajari secara pelan - pelan nanti juga perlahan - lahan mengerti.

Yang terakhir belajar GIT dan cara hosting dengan GitHub. Belajar framework dari CSS dan JavaScript. Kalian juga bisa menyiapkan builder tool untuk membantu anda, misalnya module bundlers. Lebih lengkapnya cek link dibawah ini
